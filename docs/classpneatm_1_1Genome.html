Parameters
idThe identifier of the genome
bias_schThe biases scheme (e.g., there is bias_sch[k] biases for type of index k).
inputs_schThe inputs scheme (e.g., there is inputs_sch[k] inputs for type of index k).
outputs_schThe outputs scheme (e.g., there is outputs_sch[k] outputs for type of index k).
hiddens_sch_initThe initial hidden nodes scheme (e.g., there is hiddens_sch_init[i][j] hidden nodes of input type of index i and output type of index j).
bias_valuesThe initial biases values (e.g., k-th bias will have value bias_values[k]).
resetValuesThe biases reset values (e.g., k-th bias can be resetted to resetValues[k]).
activationFnsThe activation functions (e.g., activationFns[i][j] is a pointer to an activation function that takes an input of type of index i and return a type of index j output).
conn_innovA pointer to the connections innovation tracker.
node_innovA pointer to the nodes innovation tracker.
N_ConnInitThe initial number of connections.
probRecuInitThe initial probability of recurrence.
weightExtremumInitThe initial weight extremum.
maxRecuInitThe maximum recurrence value.
loggerA pointer to the logger for logging.
