Detailed Description

template<typename T>
class pneatm::CircularBuffer< T >

A template class representing a constant size circular buffer.

A ciruclar buffer is a storing object that loop on his end to the first element. This enable it to avoid reallocating memory as it constantly overwrite previous values.
