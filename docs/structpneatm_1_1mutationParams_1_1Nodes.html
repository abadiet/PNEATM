double pneatm::mutationParams::Nodes::monotypedRate

The chance of having a monotyped node added.

Once a node is added, it has a chance of being monotyped instead of bityped. A monotyped node is a node where input and ouput types are the same. The value of rate typically lies between 0.0 and 1.0, where 0.0 indicates that a bityped will be added, and 1.0 means that a monotyped node will be added in every possible instance.

double pneatm::mutationParams::Nodes::rate

The chance of having a node added.
