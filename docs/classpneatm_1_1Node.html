Detailed Description

template<typename T_in, typename T_out>
class pneatm::Node< T_in, T_out >

A template class representing a node in a neural network.

The Node class is a template class representing a node in a neural network. It is derived from the NodeBase abstract base class and provides implementations for the virtual functions defined in the base class.

Template Parameters
T_inThe input data type for the node.
T_outThe output data type for the node.
