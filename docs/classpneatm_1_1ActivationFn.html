Detailed Description

template<typename T_in, typename T_out>
class pneatm::ActivationFn< T_in, T_out >

A template class representing an activation function in a neural network.

The ActivationFn class is a template class representing an activation function in a neural network. It is derived from the ActivationFnBase abstract base class and provides implementations for the virtual functions defined in the base class.

Template Parameters
T_inThe input data type for the activation function.
T_outThe output data type for the activation function.
