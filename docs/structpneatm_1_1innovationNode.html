Detailed Description

Structure representing the innovation tracker for nodes.

The innovationConn struct serves as an innovation tracker for nodes in a neural network. It keeps track of the innovation IDs assigned to different nodes based on input and output types, activation functions index and the node's repetition. It also provides a method to retrieve a unique innovation ID.
